5V(+),5V/(-) = adapter

- relay
	T -> 3e บอร์ด
	V -> (+) 
	G -> (-)

	COM -> (+)
	NO -> 10f บอร์ด

- water pump
	VCC สายแดง -> 10j บอร์ด
	GND สายดำ -> 17j บอร์ด

	10f บอร์ด -> NO relay
	17f บอร์ด -> (-)

- esp32
	D18 -> 3a บอร์ด
	D34 -> A0
	D4  -> 15e บอร์ด
	3V3 -> 5f บอร์ด
	GND -> (-)

- soil
	VCC -> 5g
	GND -> (-)
	A0  -> D34
- DHT11
	+   -> 5h บอร์ด
	out -> 15c บอร์ด
	-   -> (-) บอร์ด
- diode
	แถบขาว -> 10h
	แถบดำ   -> 17h
