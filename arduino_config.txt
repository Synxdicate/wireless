5V(+),5V/(-) = adapter

- relay
	T -> gpio18
	VCC -> 5V(+) สายแดง
	GND -> 5V(-) สายดำ

	COM -> 5V(+) สายแดง
	NO -> ปั้มสายแดง

- ปั้มน้ำ
	VCC สายแดง -> NO relay
	GND สายดำ -> 5V(-)

- esp32
	D18 -> 3a บอร์ด
	D34 -> A0
	D4  -> 15e บอร์ด
	3V3 -> 5f บอร์ด
	GND -> (-)

- soil
	VCC -> 5g
	GND -> (-)
	A0  -> D34
- DHT11
	+   -> 5h บอร์ด
	out -> 15c บอร์ด
	-   -> (-) บอร์ด
- diode
	แถบขาว -> 10h
	แถบดำ   -> 17h
